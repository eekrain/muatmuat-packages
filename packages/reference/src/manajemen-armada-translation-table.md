## Implementasi dalam Komponen

### 1. Import useTranslation Hook

```jsx
import { useTranslation } from "@/hooks/use-translation";
```

### 2. Gunakan dalam Komponen

```jsx
const { t } = useTranslation();

// Contoh penggunaan:
<PageTitle className="mt-4">{t('titleAddFleet')}</PageTitle>
<CardHeader className="!border-b-0 pb-4 text-xl font-semibold">
  {t('titleFleetInformation')}
</CardHeader>
```

### 3. Untuk error messages

```jsx
// Dalam validation schema
licensePlate: v.pipe(
  v.string(t('errorLicensePlateRequired')),
  v.minLength(1, t('errorLicensePlateRequired')),
  v.custom((value) => {
    const parts = value.trim().split(" ");
    return parts.length >= 2;
  }, t('errorInvalidLicensePlateFormat'))
),
```

## Label Naming Convention

### Prefix Berdasarkan Tipe Element:

- **title**: Untuk judul/header (`titleAddFleet`, `titleFleetInformation`)
- **label**: Untuk label form (`labelLicensePlate`, `labelTruckType`)
- **placeholder**: Untuk placeholder input (`placeholderLicensePlate`)
- **button**: Untuk tombol (`buttonSave`, `buttonCancel`)
- **error**: Untuk error messages (`errorLicensePlateRequired`)
- **message**: Untuk pesan umum (`messageLoading`)
- **link**: Untuk link text (`linkViewPhotoExamples`)
- **desc**: Untuk deskripsi (`descLicensePlateVisibility`)
